Aplicações Desktop

PDF Toolbox

Um utilitário desktop profissional e multiplataforma criado para a manipulação precisa de documentos PDF e Word. Desenvolvido para otimizar o controlo documental em relatórios técnicos de construção civil, inclui fusão de ficheiros (drag-and-drop), extração visual de páginas e conversão de Word para PDF em segundo plano.

A aplicação utiliza uma arquitetura Model-View-Controller (MVC), apresenta uma interface de utilizador (UI) em dark-mode totalmente desenhada no Qt Designer, e é compilada em executáveis nativos autónomos através de pipelines automatizados de CI/CD via GitHub Actions.

Python 3 PySide6 (Qt) Qt Designer PyMuPDF CI/CD

Tag de Segurança Infantil (Buddy TAG)

Um dispositivo eletrónico sem fios compacto, concebido para ajudar famílias e escolas a monitorizar a segurança das crianças. O Buddy TAG alerta os cuidadores sempre que uma criança ultrapassa uma zona de segurança predefinida, a Buddy ZONE, garantindo que as crianças se mantêm protegidas em ambientes movimentados ou de distração.

Integrado com uma aplicação móvel, permite uma configuração fácil, notificações de proximidade e tranquilidade. Desenvolvido no âmbito de um projeto de final de curso no Instituto Superior Técnico, este projeto combina eletrónica, sistemas embebidos e desenvolvimento de software para fornecer uma solução de segurança completa para menores.

Arduino Sistemas Embebidos Eletrónica IoT Segurança Infantil

Projetos Arduino

Tomada Inteligente Controlada por SMS

Um sistema embebido personalizado concebido para uma gestão remota fiável de energia via comunicação GSM. Construído com base num Arduino Nano e num módulo SIM800C, o dispositivo permite o controlo seguro de aparelhos elétricos por SMS, permitindo aos utilizadores ligar, desligar ou reiniciar a energia remotamente.

Desenvolvido com foco na fiabilidade e aplicação prática, o sistema incorpora controlo por relé, verificação de utilizadores autorizados, armazenamento persistente de configurações e uma arquitetura de firmware modular para a operação remota segura de equipamentos eletrónicos e de rede.

Arduino Nano C++ SIM800C GSM Controlo por Relé Sistemas Embebidos
