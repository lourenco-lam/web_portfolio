Desktop Applications

PDF Toolbox

A professional cross-platform desktop utility built for precision PDF and Word document manipulation. Engineered to streamline document control for technical construction reports, it features drag-and-drop merging, visual page extraction, and background Word-to-PDF conversion.

The application utilizes a Model-View-Controller (MVC) architecture, features a custom dark-mode UI designed in Qt Designer, and is compiled into standalone native executables via automated GitHub Actions CI/CD pipelines.

Parental Control

Child Safety Tag (Buddy TAG)

A compact, wireless electronic device designed to assist families and schools in monitoring children’s safety. The Buddy TAG alerts caregivers whenever a child moves beyond a pre-defined safe zone, the Buddy ZONE, ensuring children remain protected during busy or distracting environments.

Integrated with a mobile app, it allows easy setup, proximity notifications, and peace of mind. Developed as part of a final-year project at Instituto Superior Técnico, this project combines electronics, embedded systems, and software development to provide a complete safety solution for minors.

Arduino Projects

SMS-Controlled Smart Plug

A custom embedded system designed for reliable remote power management via GSM communication. Built around an Arduino Nano and SIM800C module, the device enables secure SMS-based control of electrical appliances, allowing users to remotely switch power on, off, or perform automatic restarts.

Developed with a focus on reliability and practical deployment, the system incorporates relay control, authorized user verification, persistent configuration storage, and modular firmware architecture for secure remote operation of network and electronic equipment.

Arduino Nano C++ SIM800C GSM Relay Control Embedded Systems
